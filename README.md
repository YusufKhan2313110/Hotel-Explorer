# 🏨 Hotel Explorer

Hotel Explorer is a modern and responsive hotel booking interface built with **React**, **Vite**, and **Tailwind CSS**. The application allows users to browse hotels, search and filter listings, view detailed hotel information, and save their favorite hotels using local storage.

## ✨ Features

* Browse a collection of hotels from a REST API
* Search hotels by name
* Filter hotels by rating
* View detailed information for each hotel
* Add and remove hotels from Favorites
* Favorites stored using Local Storage
* Responsive design for desktop and mobile devices
* Custom 404 Not Found page
* Fast development and optimized builds with Vite

## 🛠️ Tech Stack

* React 19
* Vite
* React Router DOM
* Tailwind CSS
* Axios
* React Icons

## 📂 Project Structure

* **Home** – Displays all hotels with search and filter functionality.
* **Hotel Details** – Shows complete information about a selected hotel.
* **Favorites** – Displays hotels saved by the user.
* **Navbar & Footer** – Shared layout components.
* **Not Found** – Handles invalid routes gracefully.

## 🚀 Getting Started

```bash
# Clone the repository
git clone <repository-url>

# Navigate into the project
cd hotel-explorer

# Install dependencies
npm install

# Start development server
npm run dev
```
